# 改进需求：AI 行动日志战术提示

> 日期：2026-04-08
> 改进点：AI 行动日志增加战术说明
> 状态：待实施

---

## 目标

AI 敌方行动时，在战斗日志中增加战术说明，让玩家理解 AI 决策逻辑，增加对战策略感。

## 现状分析

当前 `game.runAI()` 和 `game.attackAI()` 日志过于简略：
- `runAI` 中移动无日志
- 攻击只有"攻击 + 伤害"结果，无推理过程

## 改动范围

### 1. `runAI()` 增加战术日志
当 AI 寻找最佳移动位置时，日志显示：
```
[红] 黑骑士：侦察目标中...
[红] 黑骑士：向 (X,Y) 移动，准备在射程内发起攻击
```

### 2. `attackAI()` 攻击前增加战术提示
```
[红] 黑骑士：锁定目标 → 琳
[红] 琳 反击 [红] 黑骑士 → 伤害值
```

### 3. 敌方回合开始提示
```
══ 敌方回合 ══
```

## 验收标准

1. 敌方回合开始有分隔提示
2. AI 移动前有战术说明日志
3. AI 攻击前有目标锁定日志
4. 截图验证

## 改动文件

- `fe_bridge_v3.html` — `game.runAI()` / `game.attackAI()`
