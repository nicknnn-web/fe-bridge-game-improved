# 改进需求：单位详情面板增强

> 日期：2026-04-08
> 改进点：单位详情面板信息层次优化
> 状态：待实施

---

## 目标

优化单位详情面板的信息展示，让属性层次更清晰、视觉更精致。

## 现状分析

当前 `UI.showUnit()` 面板（line 764-802）：
- HP 只显示数值，无图标
- 六维属性网格只有文字标签，无图标
- 地形信息独立一栏，视觉上与属性脱节

## 改动范围

### 1. HP 显示增加图标
当前：`HP  18 / 22`
改为：`❤️ HP  18 / 22`

### 2. 属性标签增加小图标
| 属性 | 图标 |
|------|------|
| 攻击 | ⚔ |
| 防御 | 🛡 |
| 移动 | 👟 |
| 速度 | 💨 |
| 射程 | 🎯 |
| 最大HP | ❤️ |

### 3. 防御值显示优化
当有地形加成时，`4+2` 格式更明显：
当前：防御 `4+2`
改为：防御 `4` + `(+2地形)`

### 4. 状态标签加图标
当前：`待命中`
改为：`● 待命中`

### 5. 已行动单位降低面板饱和度
acted=true 时，面板整体加一层半透明暗色遮罩，视觉提示"这个单位本回合已结束"。

## 验收标准

1. HP 条旁有 ❤️ 图标
2. 每个属性标签前有对应图标
3. 地形防御加成格式清晰
4. 截图验证

## 改动文件

- `fe_bridge_v3.html` — `UI.showUnit()` 函数
